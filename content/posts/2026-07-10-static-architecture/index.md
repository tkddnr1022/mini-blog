M3 UI 검증을 위해 추가한 두 번째 샘플 포스트입니다. 이 글은 `Notes` 카테고리에 속하며, 더 이른 날짜로 정렬되어 이전/다음 글 링크를 테스트합니다.

## 아키텍처 개요

Next.js App Router와 정적 export를 사용해 GitHub Pages에 배포합니다.

### 빌드 파이프라인

마크다운은 빌드 시점에 HTML로 변환되고, 이미지는 public 경로로 복사됩니다.

## 이미지 없는 짧은 섹션

라이트박스가 없는 포스트에서도 TOC와 네비게이션이 정상 동작하는지 확인합니다.
